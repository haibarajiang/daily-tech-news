## 3. 工具与深读

- [Perry：将 TypeScript 直接编译为可执行文件](https://www.perryts.com/) (Hacker News)
  Perry 是一款新工具，利用 SWC 和 LLVM 将 TypeScript 代码直接编译为本地可执行文件。这为 TypeScript 开发者提供了一种更轻量、无需 Node.js 运行时的分发方案，有助于提升应用启动速度和部署便捷性。

- [Tiny-vLLM：高性能 LLM 推理引擎](https://github.com/jmaczan/tiny-vllm) (Hacker News)
  这是一个使用 C++ 和 CUDA 编写的高性能大模型推理引擎。项目旨在提供轻量级但高效的推理解决方案，适合需要深度定制或资源受限环境的开发者，是对 Python 生态主流推理框架的有力补充。

- [SQLite is all you need for durable workflows](https://obeli.sk/blog/sqlite-is-all-you-need-for-durable-workflows/) (Hacker News)
  文章深入探讨了如何利用 SQLite 实现持久化工作流。作者提出了一种基于 SQLite 事务特性的架构方案，能够以极低的依赖成本实现高可靠的任务调度与状态管理，为构建轻量级后端系统提供了新思路。

- [PrusaSlicer 推出开源 ColorMix 多色打印方案](https://blog.prusa3d.com/our-new-open-source-colormix-model-in-prusaslicer-and-easyprint_136079/) (Hacker News)
  Prusa 发布了新的开源 ColorMix 模型，支持在 PrusaSlicer 中进行数十种颜色的混合打印。该功能降低了多色 3D 打印的门槛，让普通用户也能利用单一喷头实现复杂的色彩效果，极具实用价值。

- [什么才是「好」的 Android 音频输出？](https://sspai.com/post/110120) (少数派)
  文章从技术原理出发，探讨了 Android 高保真音频输出的实现难点。作者分析了系统音频处理链路对音质的影响，并分享了如何通过设置和优化，在 Android 设备上获得更纯净的音频体验，适合发烧友阅读。
