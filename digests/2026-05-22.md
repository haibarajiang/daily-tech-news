- [三星芯片工人平均获 34 万美元奖金，AI 利润飙升](https://qz.com/samsung-chip-workers-bonus-ai-profits-052126) (Hacker News)

- [SpaceX 计划五年内实现每年 1 万次发射](https://36kr.com/p/3816942732133510) (36kr)
  SpaceX 向美国联邦航空管理局（FAA）透露了宏伟计划，目标是在五年内将发射频率提升至每年 1 万次。该计划旨在支持其卫星星座的组建，但 FAA 表示需先证明其可靠性提升，这一目标若实现将彻底改变人类进入太空的频率与成本。

## 2. 趋势与解读

- [Agent 拉动 Token 生意，阿里云找到商业化突破口](https://36kr.com/p/3819955387371649) (36kr)
  阿里云 Token 收入的爆发式增长，核心动力在于 Agent 应用的落地。这表明大模型行业的商业模式正发生质变，企业不再满足于简单的模型 API 调用，而是通过构建具备自主规划能力的 Agent 解决具体业务问题，这将是 MaaS 落地的关键路径。

- [如何消除 vibe code 产生的技术债？](https://www.v2ex.com/t/1214452) (V2EX)
  随着 AI 辅助编程的普及，“vibe coding”（氛围编程）虽提升了效率，但也产生了大量难以维护的“屎山代码”和技术债。社区讨论指出，开发者不能完全依赖 AI 生成，仍需具备代码审查和架构把控能力，否则后续的维护成本将抵消 AI 带来的效率红利。

- [具身智能的数据供给革命：质量比数量更重要](https://36kr.com/p/3819884983226497) (36kr)
  与大语言模型依赖海量数据不同，具身智能（机器人）面临的是动态物理世界，单纯堆砌数据难以训练出可靠模型。行业正在转向结构化、高质量的数据供给管线，通过五层数据编译模型解决时序、空间和任务意图的耦合问题，这是机器人进工厂的前提。

- [AMD 预计未来几年 CPU 需求将持续增长](https://36kr.com/p/3819887120306564) (36kr)
  AMD CEO 苏姿丰表示 CPU 市场增长超预期，未来五年年增长率将超 35%。这打破了“AI 时代只需 GPU”的刻板印象，实际上 AI 应用的推理端和传统算力需求依然强劲，CPU 与 GPU 的协同进化仍是算力基础设施发展的主旋律。

- [Waymo 暂停亚特兰大服务，Robotaxi 频频驶入洪水](https://techcrunch.com/2026/05/21/waymo-pauses-atlanta-service-as-its-robotaxis-keep-driving-into-floods/) (Hacker News)
  Waymo 因 Robotaxi 在洪水环境中未能正确识别危险并持续行驶而暂停了亚特兰大服务。该事故暴露了自动驾驶在极端天气和长尾场景下的感知短板，也提醒行业：自动驾驶的安全冗余不仅仅是技术指标，更是商业运营的生命线。
